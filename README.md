# Vue3 Theme

> Create a vue3 theme using router and bulma css.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Preview for production
```
yarn preview
```

### Lints and fixes files
```
yarn lint
```
